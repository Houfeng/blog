如何使用？

解析(jtp.parse)

代码:

var html='<div>My name is <@$(this.name)@></div>';
var rs=jtp.parse(html,{name:'jtp'});

结果: 

<div>My name is jtp</div>

编译(jtp.complite)

代码:

var html='<div>My name is <@$(this.name)@></div>';
var fn=jtp.complite(html);
var rs=fn({name:'jtp'});

结果:

<div>My name is jtp</div>
